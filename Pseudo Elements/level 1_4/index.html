Pseudo-Elemente in CSS sind imaginäre Elemente, die über :: (z. B. ::before, ::after) an ein Element angehängt werden. Sie ermöglichen das Hinzufügen von dekorativen Inhalten oder das Stylen spezifischer Teile eines Elements, ohne HTML zu ändern. Der Hauptunterschied besteht darin, dass Pseudo-Elemente bestimmte Teile eines Elements ansprechen, die nicht durch das eigentliche HTML-Markup existieren. Sie ermöglichen das Hinzufügen von zusätzlichem Inhalt oder das Stylen spezifischer Teile ohne das Markup zu ändern, im Gegensatz zum normalen Styling, das auf vorhandenen HTML-Elementen basiert.
